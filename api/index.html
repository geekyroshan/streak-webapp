GitHub Streak Manager API

This is the API server for the GitHub Streak Manager application.

Health Check

Path: /api

Method: GET

Returns basic API status information.

GitHub Authentication

Path: /api/auth/github

Method: GET

Initiates the GitHub OAuth authentication flow.

Note: Please access the main application at the root URL instead of this API documentation page.

The API is primarily designed for the frontend application to consume and not for direct use.
